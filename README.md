# Todo App - Flask

Sistema de gerenciamento de tarefas desenvolvido com **Flask + SQLAlchemy + HTML/JavaScript**.  
O projeto permite criar, editar, concluir, excluir e filtrar tarefas com persistência em banco de dados.

---

 Funcionalidades

- ✔ Criar tarefas com título, descrição e data de vencimento  
- ✔ Listar todas as tarefas  
- ✔ Editar tarefas existentes  
- ✔ Marcar tarefas como concluídas  
- ✔ Excluir tarefas  
- ✔ Filtrar tarefas (pendentes / concluídas / todas)  
- ✔ Interface simples integrada com API REST  

---

Tecnologias utilizadas

- Python 3
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML5
- JavaScript (Fetch API)
- CSS básico (estrutura simples)
